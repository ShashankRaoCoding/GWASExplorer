How to use GWAS Explorer



GWAS Explorer is meant for in-browser visualisation of data, primarily summary statistics (although other data will also likely work). To use, simply:

  1. Select the files you would like to visualise
  2. Select the parameters to plot
  3. Select the id (this is to help you identify your data points)
Hover the cursor over the data points to see more info. None of the data is sent to our servers. Coming soon: + Click on the data points to see in depth SNP analysis (if rsids present) from automated GNOMAD database querying.

This site is NOT the GWAS Explorer hosted by the NCI.
